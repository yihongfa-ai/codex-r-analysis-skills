# 数据分析方法库与模型路由表

## 使用方式

本文件用于在 `exploratory-r-analysis` 和 `ssci-r-analysis-writing` 中选择、排除和记录统计方法。不要一次性运行所有方法；先按研究目的、数据结构、因变量类型、推断强度和工作流阶段做路由。

文本量化边界：文本清洗、分词、词典计分、主题模型、嵌入、文本分类、情感分析等默认交给 Python 或文本分析工作流完成。R 端只接收 Python 写回数据文件的结构化变量，并记录变量定义、生成来源、验证证据和量表方向。

## 总分类框架

| 维度 | 分类 | 决策用途 |
|---|---|---|
| 研究目的 | 描述、测量、关联、预测、因果、机制、配置、网络生成、稳健性 | 决定模型能回答什么问题 |
| 数据结构 | 横截面、嵌套/多层、面板/纵向、时间序列、网络、空间、复杂抽样、元分析、Python 派生文本变量 | 决定可用模型族 |
| 因变量类型 | 连续、二元、序数、多分类、计数、比例、组成、时长/生存、潜变量、类别/画像、网络边 | 决定估计函数和诊断 |
| 推断强度 | 描述性、相关性、预测性、因果性、机制性、探索性筛查 | 决定报告措辞和证据边界 |
| 工作流阶段 | 清洗筛查、候选模型图、探索运行、正式建模、稳健性、论文报告 | 决定输出深度 |

## 1. 描述统计、数据筛查与量表基础

- 适用问题：样本是什么、变量分布如何、缺失是否严重、量表是否可靠。
- 数据结构：多数结构化数据。
- 推荐 R 包：`dplyr`, `skimr`, `naniar`, `psych`, `gtsummary`, `modelsummary`, `ggplot2`。
- 诊断检查：缺失模式、极端值、低方差、量表方向、信度、分布偏态。
- 必要输出：变量字典、缺失表、描述统计表、相关表、量表信度表。
- 不适用情形：不能把描述差异直接写成因果效应。

## 2. 常规回归与 GLM

- 适用问题：连续、二元、序数、计数或比例型结果的关联检验。
- 数据结构：横截面或清楚定义的分析样本。
- 推荐 R 包：`stats`, `broom`, `sandwich`, `lmtest`, `car`, `marginaleffects`, `ggeffects`, `ordinal`, `MASS`, `pscl`, `betareg`, `glmmTMB`。
- 诊断检查：残差、异方差、共线性、过度离散、零膨胀、比例优势假设、影响点。
- 必要输出：模型公式、样本量、系数、置信区间、稳健或聚类标准误、边际效应。
- 常见误区：把 Likert 单题机械当作连续变量但没有敏感性分析。

## 3. 交互、调节、非线性与边际效应

- 适用问题：效应是否因群体、情境或资源水平而变化。
- 推荐 R 包：`marginaleffects`, `ggeffects`, `interactions`, `splines`, `mgcv`。
- 诊断检查：连续变量中心化、简单斜率、Johnson-Neyman 区间、预测值范围、过度外推。
- 必要输出：交互项、边际效应表、预测值图、简单斜率或条件效应。
- 常见误区：只解释交互项显著性，不解释实质效应大小。

## 4. 缺失数据与多重插补

- 适用问题：缺失比例或缺失机制可能影响结论。
- 推荐 R 包：`mice`, `miceadds`, `Amelia`, `naniar`, `VIM`。
- 关键要求：区分结构性缺失、跳题缺失、随机缺失和潜在非随机缺失。
- 诊断检查：缺失模式、插补模型收敛、插补后分布、完整案例对照。
- 必要输出：缺失机制说明、插补变量清单、插补次数、合并规则、敏感性结果。
- 不适用情形：关键变量缺失完全由设计跳题导致且不应插补。

## 5. 复杂抽样、权重与调查设计

- 适用问题：数据带有抽样权重、分层、PSU、重复权重或总体推断目标。
- 推荐 R 包：`survey`, `srvyr`, `weights`, `jtools`。
- 关键要求：明确权重类型，区分抽样权重、后分层权重、频数权重和分析权重。
- 诊断检查：权重分布、极端权重、设计效应、加权/未加权结果对照。
- 必要输出：设计对象、权重变量、分层和 PSU、加权描述统计、设计校正标准误。
- 常见误区：把抽样权重直接塞进普通 `lm()` 而不处理设计方差。

## 6. 心理测量、IRT、潜类别与测量不变性

- 适用问题：问卷量表、潜在构念、题项质量、跨群体可比性。
- 推荐 R 包：`psych`, `lavaan`, `semTools`, `mirt`, `TAM`, `eRm`, `poLCA`, `tidyLPA`, `mclust`。
- 诊断检查：信度、因子载荷、模型拟合、DIF、阈值、类别数、熵、群体不变性。
- 必要输出：题项表、载荷/参数表、拟合指数、类别解释、测量不变性步骤。
- 常见误区：未检验测量结构就直接合成量表并做强解释。

## 7. SEM、CFA、路径模型与中介

- 适用问题：潜变量测量、路径假设、理论机制或中介模型。
- 推荐 R 包：`lavaan`, `semTools`, `semPlot`, `blavaan`。
- 诊断检查：拟合指数、标准化载荷、残差、修改指数、间接效应置信区间、样本量。
- 必要输出：测量模型、结构模型、拟合表、直接/间接/总效应、路径图。
- 不适用情形：横截面中介只能谨慎写为机制一致性或探索性证据。

## 8. 多层、纵向与成长模型

- 适用问题：学生嵌套学校、员工嵌套组织、重复测量、纵向变化。
- 推荐 R 包：`lme4`, `lmerTest`, `glmmTMB`, `nlme`, `lcmm`, `coxme`。
- 诊断检查：ICC、随机效应结构、收敛、残差、层级样本量、随机斜率稳定性。
- 必要输出：层级结构、固定效应、随机效应、方差分量、ICC、预测变化图。
- 常见误区：聚类很强却用普通 OLS 且不校正标准误。

## 9. 面板、DiD、IV、RDD 与现代因果推断

- 适用问题：有明确处理、时间顺序、工具变量、断点或准实验设计。
- 推荐 R 包：`fixest`, `plm`, `did`, `AER`, `ivreg`, `rdrobust`, `MatchIt`, `WeightIt`, `cobalt`, `sensemakr`, `grf`, `DoubleML`, `drtmle`, `Synth`, `augsynth`。
- 诊断检查：平行趋势、弱工具、断点操纵、平衡性、重叠性、敏感性、聚类标准误。
- 必要输出：识别假设、设计图、主估计、诊断表、稳健性、安慰剂或敏感性分析。
- 常见误区：只有控制变量时就使用因果语言。

## 10. 时间序列与中断时间序列

- 适用问题：单个或多个单位随时间变化、政策冲击、预测或动态关系。
- 推荐 R 包：`forecast`, `fable`, `tsibble`, `vars`, `tseries`, `urca`, `nlme`。
- 诊断检查：平稳性、自相关、季节性、结构断点、残差白噪声、预测误差。
- 必要输出：时间图、ACF/PACF、模型参数、诊断图、预测或干预效应图。
- 不适用情形：时间点太少无法估计动态结构。

## 11. 空间与地理模型

- 适用问题：地理单元、空间邻近、地区扩散或空间依赖。
- 推荐 R 包：`sf`, `spdep`, `spatialreg`, `tmap`, `ggplot2`。
- 诊断检查：邻接矩阵定义、Moran's I、空间滞后/误差依赖、地图投影。
- 必要输出：空间权重说明、空间自相关检验、空间模型表、地图或残差地图。
- 常见误区：忽略空间依赖导致标准误和解释偏误。

## 12. 网络分析与 ERGM/TERGM

- 适用问题：关系边、网络结构、节点属性与关系生成机制。
- 推荐 R 包：`igraph`, `tidygraph`, `ggraph`, `network`, `sna`, `statnet`, `ergm`, `tergm`, `btergm`。
- 诊断检查：网络边界、边含义、密度、互惠、三角闭合、收敛、退化、GOF。
- 必要输出：网络描述、中心性/社群、模型项、收敛诊断、GOF 图。
- 常见误区：把非独立的网络边当作普通独立观测。

## 13. QCA 与集合关系

- 适用问题：中小样本配置、充分/必要条件、集合成员关系。
- 推荐 R 包：`QCA`, `SetMethods`。
- 诊断检查：校准锚点、必要性、一致性、覆盖度、频数阈值、反事实方向。
- 必要输出：校准表、真值表、必要性/充分性结果、中间解/简约解/复杂解。
- 常见误区：校准没有理论依据，只按分位数机械切分。

## 14. 元分析

- 适用问题：多个研究或效应量的综合、异质性和发表偏倚评估。
- 推荐 R 包：`metafor`, `meta`, `clubSandwich`。
- 诊断检查：效应量编码、依赖效应量、异质性、影响点、漏斗图、小样本校正。
- 必要输出：效应量数据字典、森林图、异质性统计、调节分析、偏倚检查。
- 不适用情形：来源研究不可比或效应量无法可靠转换。

## 15. 贝叶斯模型

- 适用问题：小样本不确定性、层级结构、先验信息、复杂模型或完整后验报告。
- 推荐 R 包：`brms`, `rstanarm`, `blavaan`, `bayestestR`, `posterior`。
- 诊断检查：先验合理性、Rhat、ESS、后验预测检查、链收敛、敏感性。
- 必要输出：先验说明、后验均值/中位数、可信区间、诊断、后验预测图。
- 常见误区：只把贝叶斯模型当作显著性替代品。

## 16. 可解释机器学习

- 适用问题：预测、变量筛查、非线性关系探索或高维特征压缩。
- 推荐 R 包：`tidymodels`, `glmnet`, `ranger`, `xgboost`, `mgcv`, `vip`, `DALEX`, `iml`, `fastshap`, `kernelshap`, `shapviz`, `pdp`。
- 诊断检查：训练/测试划分、交叉验证、过拟合、校准、变量重要性稳定性、PDP/ICE 外推。
- 必要输出：性能指标、调参记录、变量重要性、解释图、与可解释基线模型对照。
- 报告边界：默认写作预测性或探索性证据，不写成因果证据。

## 17. 特殊因变量与特殊数据形态

- 适用问题：零膨胀计数、比例/率、组成数据、半连续结果、联合纵向-生存数据。
- 推荐 R 包：`glmmTMB`, `pscl`, `betareg`, `DirichletReg`, `compositions`, `survival`, `JMbayes2`。
- 诊断检查：边界值、零膨胀、过度离散、组成闭合约束、删失机制。
- 必要输出：分布选择理由、替代规格、预测均值/概率、敏感性分析。
- 常见误区：把比例变量直接 OLS 而不检查边界和异方差。

## 18. 稳健性、敏感性、多重检验与功效

- 适用问题：检验结论是否依赖模型选择、样本筛选、控制变量或多重检验。
- 推荐 R 包：`sensemakr`, `TOSTER`, `specr`, `DeclareDesign`, `pwr`, `simr`, `clubSandwich`。
- 诊断检查：多重检验 FDR、等效性、规格曲线、影响点、控制块、功效或最小可检测效应。
- 必要输出：稳健性矩阵、规格曲线、敏感性参数、功效说明、结论边界。
- 常见误区：只展示支持主结论的稳健性，不报告失败或边界条件。

## 输出路由建议

| 阶段 | exploratory-r-analysis | ssci-r-analysis-writing |
|---|---|---|
| 清洗与筛查 | 发现问题、提出候选处理 | 锁定正式处理规则 |
| 方法选择 | 生成候选模型图并分类 | 根据研究设计选择正式模型 |
| 高级方法 | 判断可行性，跑小规模或候选模型 | 完整估计、诊断、稳健性和报告 |
| 图表 | 中文探索图、模型比较图 | 论文表格、正式图、英文 Methods/Results |
| 文本变量 | 等待 Python 输出变量后探索 | 使用 Python 变量写入正式模型与报告 |
